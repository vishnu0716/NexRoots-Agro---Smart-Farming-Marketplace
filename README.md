 NexRoots Agro - Smart Farming Marketplace
 📌 Overview
NexRoots Agro is a modern, responsive agricultural e-commerce landing page designed to empower farmers with easy access to premium farming products including herbicides, growth promoters, fungicides, and high-quality seeds. The website showcases a clean, trustworthy interface that bridges the gap between agri-tech solutions and rural farming communities.

✨ Features
🎨 Modern & Responsive Design - Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
🌾 Product Showcase - Display of four main product categories with pricing and descriptions
📊 Live Statistics - Dynamic counter section showing farmers served, satisfaction rate, and villages covered
🧪 Interactive Cards - Clickable product cards with alert functionality for product inquiries
🌟 Testimonial Section - Real farmer reviews with star ratings
🚚 Feature Highlights - Free delivery, soil testing kit offer, and 24/7 expert support
📧 Newsletter Subscription - Email/mobile input for offers and updates
🔐 Farmer Login Portal - Placeholder for upcoming farmer dashboard
🎯 Smooth Scrolling - Navigation enhancements for better UX

🛠️ Technologies Used
HTML5 - Semantic structure
CSS3 - Flexbox, Grid, animations, gradients, backdrop filters
JavaScript - Interactive elements, smooth scrolling, event handling
Font Awesome 6 - Icons for visual enhancement
Google Fonts - Inter font family for modern typography

🚀 Getting Started
Prerequisites
Any modern web browser (Chrome, Firefox, Safari, Edge)
Text editor (VS Code, Sublime Text, etc.) for modifications
Installation
Clone the repository

Simply open index.html in your browser
Or use Live Server in VS Code for better development experience

No Dependencies Required!
The project uses CDN links for fonts and icons, so an internet connection is required for full icon and font rendering.
